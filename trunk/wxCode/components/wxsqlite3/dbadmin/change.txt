Version 0.2.3
* Correction bogue sur dialogue filtre, liste de champs parfois incomplète.
* Correction bogue sur affichage des colonnes avec type WXSQLITE_NULL.
* Correction bogue sur affichage de la taille des bouttons dans la wxToolBar
  l'onglet DATA de l'explorateur dans la version Windows.
* Correction bogue exception levée lors du raffraichissement des colonnes
  dans la fenêtre d'affichage de la structure de la base de données.
* Gestion/Affichage des valeurs nulles
* Correction bogue taille de la fenêtre des paramètres SQL
* Sauvegarde/restauration de la valeur des paramètres SQL
* Gestion des bases chiffrées EAS (RSA Data Security license)

Version 0.2.2
* Première version sous GNU/Linux (correction des bogues sous wxGTK2).
* Ajout d'un élément de menu contextuel database.
* Gestion de fin de ligne de commande SQL par point virgule au lieu de ligne vide.
* Correction de la non prise en compte des commentaires sur les fins de ligne
  dans l'éditeur SQL pour la détermination de la commande courrante.
* Correction de l'implémentation de la boite de dialogue préférences.
* Remplacement du gestionnaire d'événement wxEVT_LEFT_DCLICK par 
  EVT_TREE_ITEM_ACTIVATED et correction du bogue sur la gestion du flag de HitTest
* Ajout de possibilité de filtre (clause WHERE) sur l'onglet Data de l'explorateur.
* Ajout de possibilité de true (clause ORDER BY) sur l'onglet Data de l'explorateur.
* Correction d'un bogue de raffraichissement de l'arbre si aucun élément sélectionnée.
* Correction d'un bogue sur l'affichage des coches dans les grilles de l'explorateur.

Version 0.2.0
* Gestion de plusieurs bases (ATTACH/DETACH DATABASE).
